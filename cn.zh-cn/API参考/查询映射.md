# 查询映射<a name="ZH-CN_TOPIC_0110485097"></a>

## 功能介绍<a name="section544449010253"></a>

该接口用于查询映射信息。

该接口仅适用于全局区域（ALL），全局区域的域名为iam.myhuaweicloud.com。

## URI<a name="section961061110253"></a>

-   URI格式

    GET /v3/OS-FEDERATION/mappings/\{id\}
